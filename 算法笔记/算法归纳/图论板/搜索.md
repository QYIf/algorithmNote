# 搜索

## 深度优先搜索（DFS）

```cpp
void dfs(/*所需记录的条件*/) {
    if(/*搜索完毕的条件*/)	{}
    if(/*剪枝*/) {}
    if(/*成功的条件*/) {}
    /*针对不同的方向进行搜索*/
    for(int i = 0; i < /*方向的数量*/; i++) {
        if(/*符合条件（没被选过或走过）*/) {
            /*选入答案或执行某个操作*/
            dfs(/*继续往下进行搜索*/);
            /*回溯*/
        }
    }
}
```
