# C++STL使用

## 1.stack
>> #### 1. 创建
>> stack<类型>;
>
>> 2. 操作
>>     + empty (判断是否为空)
>>     + size (返回栈的元素个数)
>>     + back (返回栈顶元素)
>>     + push_back (入栈)
>>     + pop_back (出栈)

## 2. queue
>> #### 1. 创建
>> queue<类型>;
> 
>> 2. 操作
>>     + empty (判断是否为空)
>>     + size (返回栈的元素个数)
>>     + back (返回栈顶元素)
>>     + push_back (入栈)
>>     + pop_back (出栈)
